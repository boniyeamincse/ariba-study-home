# Contributing to Ariba Study Home

First off, thank you for considering contributing to Ariba Study Home! It's people like you that make Ariba Study Home such a great tool.

## Code of Conduct

By participating in this project, you are expected to uphold our Code of Conduct. Please report unacceptable behavior to the project maintainers.

## How Can I Contribute?

### Reporting Bugs

- Use a clear and descriptive title for the issue.
- Describe the exact steps which reproduce the problem.
- Provide screenshots or recordings if applicable.

### Suggesting Enhancements

- Use a clear and descriptive title.
- Provide a step-by-step description of the suggested enhancement.
- Explain why this enhancement would be useful to most users.

### Pull Requests

- Ensure your code follows the existing style of the project.
- Update the documentation with details of changes to the interface.
- Perform self-tests to ensure no existing functionality is broken.

## Style Guide

- Use semantic HTML tags.
- Keep CSS variables organized in `:root`.
- Write clean, commented JavaScript code.
